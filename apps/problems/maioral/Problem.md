# Maioral

## Descrição

Dada uma lista de inteiros, escreva uma função para encontrar o elemento máximo na lista.

## Exemplo

### Exemplo 1

**Entrada:** `[3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]`  
**Saída:** `9`

### Exemplo 2

**Entrada:** `[10, 20, 30, 40, 50]`  
**Saída:** `50`

## Restrições

- `1 <= lista.length <= 10^4`
- `-10^6 <= lista[i] <= 10^6`
